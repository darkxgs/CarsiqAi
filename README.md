## 📋 نظرة عامة

<div dir="rtl">

**CarsiqAi** هو تطبيق ذكي ثوري مصمم خصيصاً لمساعدة أصحاب السيارات في العراق على اتخاذ قرارات مدروسة حول زيوت سياراتهم. يجمع التطبيق بين قوة الذكاء الاصطناعي وخبرة متخصصة في صناعة السيارات لتقديم توصيات دقيقة ومخصصة تناسب الظروف المناخية الصعبة في العراق.

### 🎯 لماذا CarsiqAi؟

في العراق، تواجه السيارات تحديات فريدة:
- **درجات حرارة قاسية** تصل إلى 52 درجة مئوية صيفاً
- **عواصف ترابية شديدة** تؤثر على أداء المحرك
- **تباين في جودة الوقود** المتوفر في الأسواق
- **طرق وعرة** تتطلب عناية خاصة بالمحرك

**CarsiqAi** يحل هذه التحديات من خلال توصيات علمية مدروسة!

</div>

## 🛠️ التقنيات والأدوات المستخدمة

<div dir="rtl">

### 🏗️ البنية التحتية
- **Next.js 14** - إطار عمل React متطور للأداء العالي
- **React 18** - أحدث إصدار من مكتبة React للواجهات التفاعلية
- **TypeScript** - لغة برمجة آمنة النوع للتطوير الموثوق

### 🎨 التصميم والواجهة
- **Tailwind CSS** - إطار CSS مرن للتصميم السريع
- **shadcn/ui** - مكتبة مكونات UI عصرية وقابلة للتخصيص
- **CSS Modules** - لتنظيم الأنماط وإدارتها بكفاءة

### 🤖 الذكاء الاصطناعي
- **OpenRouter API** - منصة متقدمة للوصول لأفضل نماذج الذكاء الاصطناعي
- **Vercel AI SDK** - أدوات متطورة لدمج تقنيات الذكاء الاصطناعي
- **Streaming Responses** - استجابات فورية وسلسة

### 📊 البيانات والتخزين
- **CarQuery API** - قاعدة بيانات شاملة لمواصفات السيارات
- **Local Storage** - تخزين محلي آمن للمحادثات والتفضيلات
- **State Management** - إدارة ذكية لحالة التطبيق

## ⚙️ التثبيت والتشغيل المحلي

<div dir="rtl">

### 📋 المتطلبات المسبقة
- **Node.js** (الإصدار 18 أو أحدث)
- **npm** أو **yarn** لإدارة الحزم
- **مفتاح API من OpenRouter** ([احصل عليه من هنا](https://openrouter.ai/))

### 🚀 خطوات التثبيت

</div>

```bash
# 1. استنساخ المشروع
git clone https://github.com/darkxgs/CarsiqAi.git
cd CarsiqAi

# 2. تثبيت الاعتمادات
npm install
# أو باستخدام yarn
yarn install

# 3. إنشاء ملف متغيرات البيئة
cp .env.example .env.local

# 4. تحرير ملف .env.local وإضافة مفتاح API
# OPENROUTER_API_KEY=your_openrouter_api_key_here

# 5. تشغيل المشروع للتطوير
npm run dev
# أو
yarn dev

# 6. افتح المتصفح على العنوان
# http://localhost:3000
```

<div dir="rtl">

### 📁 هيكل ملف `.env.local`

</div>

```env
# مفتاح API من OpenRouter (مطلوب)
OPENROUTER_API_KEY=your_openrouter_api_key_here

# إعدادات اختيارية
NEXT_PUBLIC_APP_NAME=CarsiqAi
NEXT_PUBLIC_APP_VERSION=1.0.0
```
