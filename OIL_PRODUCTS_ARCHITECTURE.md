## Database Schema

```sql
CREATE TABLE oil_product (
  id            TEXT PRIMARY KEY,
  name          TEXT NOT NULL,
  brand         TEXT NOT NULL,
  product_line  TEXT NOT NULL,
  viscosity     TEXT NOT NULL,
  type          TEXT NOT NULL, -- FULL_SYNTHETIC, SEMI_SYNTHETIC, MINERAL
  api_spec      TEXT,
  acea_spec     TEXT,
  other_specs   TEXT[],
  capacity      FLOAT,
  price         FLOAT,
  stock         INTEGER DEFAULT 0,
  is_active     BOOLEAN DEFAULT true,
  description   TEXT,
  image_url     TEXT,
  compatible_for TEXT[], -- ['american', 'european', 'asian']
  created_at    TIMESTAMP DEFAULT NOW(),
  updated_at    TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_oil_product_brand ON oil_product(brand);
CREATE INDEX idx_oil_product_viscosity ON oil_product(viscosity);
CREATE INDEX idx_oil_product_is_active ON oil_product(is_active);
```

## Matching Algorithm

```javascript
function scoreProduct(product, requirement) {
  let score = 100
  
  // Exact viscosity match (required)
  if (product.viscosity === requirement.viscosity) {
    score += 50
  } else {
    return 0 // No match
  }
  
  // Type match
  if (product.type === requirement.type) {
    score += 30
  }
  
  // API spec match
  if (requirement.apiSpec && product.apiSpec?.includes(requirement.apiSpec)) {
    score += 20
  }
  
  // ACEA spec match
  if (requirement.aceaSpec && product.aceaSpec?.includes(requirement.aceaSpec)) {
    score += 20
  }
  
  // Stock availability
  if (product.stock > 0) {
    score += 10
  }
  
  return score
}
```

## Brand Priority Logic

```javascript
function getBrandPriority(carType) {
  switch (carType) {
    case 'american':
      return ['Valvoline', 'Castrol']
    
    case 'european':
      return ['Liqui Moly', 'Meguin']
    
    case 'asian':
      return ['Valvoline', 'Castrol', 'Liqui Moly', 'Meguin']
    
    default:
      return ['Castrol', 'Liqui Moly', 'Valvoline', 'Meguin']
  }
}
```

## Car Type Detection

```javascript
function getCarType(brand) {
  const american = ['ford', 'jeep', 'chevrolet', 'dodge', 'cadillac', 'gmc']
  const european = ['mercedes', 'bmw', 'audi', 'volkswagen', 'porsche', 'volvo']
  
  if (american.includes(brand.toLowerCase())) return 'american'
  if (european.includes(brand.toLowerCase())) return 'european'
  return 'asian' // Default
}
```

## Security Considerations

1. **Admin Authentication**: Required for product management
2. **Input Validation**: All API inputs validated with Zod
3. **SQL Injection**: Protected by Prisma ORM
4. **XSS Protection**: React auto-escapes output
5. **Rate Limiting**: Consider adding for API endpoints

## Performance Optimizations

1. **Database Indexes**: On brand, viscosity, isActive
2. **Query Optimization**: Filtered queries with specific WHERE clauses
3. **Caching**: Consider Redis for frequently accessed products
4. **Pagination**: For large product lists in admin
5. **Lazy Loading**: Load products on-demand

## Scalability

- **Horizontal Scaling**: Stateless API design
- **Database**: PostgreSQL with connection pooling
- **CDN**: For product images
- **Caching**: Redis for hot data
- **Load Balancing**: Multiple API instances
